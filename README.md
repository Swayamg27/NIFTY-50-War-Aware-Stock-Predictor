# NIFTY-50-War-Aware-Stock-Predictor
Machine learning model predicting NIFTY 50 market direction using 29 features such as technical indicators, VIX, crude oil, USD/INR, CPI, Repo Rate, and geopolitical sentiment scores covering COVID, Russia-Ukraine, Israel-Gaza, US-Iran, and US-China trade war events. Random Forest | Python | scikit-learn
